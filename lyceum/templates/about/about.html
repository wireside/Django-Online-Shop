О проекте "Lyceum"

Lyceum — это учебный проект интернет-магазина, разработанный на фреймворке Django. Цель проекта — показать возможности современного веб-разработки и создать функциональную платформу для демонстрации электронных продаж.

Цели проекта

  • Создать удобный и функциональный интернет-магазин.
  • Применить современные подходы к разработке (CI/CD, адаптивный дизайн).
  • Научиться работать с базами данных, моделями и пользователями.

Функционал

  • Каталог товаров с фильтрацией по категориям и тегам.
  • Корзина покупок с сохранением между сессиями.
  • Регистрация и авторизация пользователей.
  • Интеграция с админ-панелью для управления данными.

Технологии

Проект построен с использованием следующих технологий:

  • Backend: Python, Django
  • Frontend: HTML, CSS, Bootstrap
  • База данных: SQLite
  • Автоматизация: GitHub Actions
  • Контроль версий: Git

Будущие улучшения

В будущем планируется добавить:

  • Интеграцию с платёжными системами (например, Stripe или PayPal).
  • Систему отзывов и рейтингов для товаров.
  • Поддержку личного кабинета пользователя с историей заказов.
